# 🛒 Mini E-Commerce Frontend (React.js)

A modern mini e-commerce web application built using **React + Vite**, featuring product browsing, cart management, and an admin panel. This project uses a **dummy server/mock data** instead of a real backend.

---

## 🚀 Features

* 🏬 Product listing page
* 🔍 Product details view
* 🛒 Add to cart / remove from cart
* ➕ Update product quantity
* 💳 Cart summary & pricing
* 🧑‍💼 Admin panel (manage products - UI only)
* ⚡ Fast performance with Vite
* 📦 State management using Redux
* 📦 Order history & tracking
* 🔐 Authentication system (Login / Signup)
* 💳 Payment integration

---

## 🛠️ Tech Stack

* ⚛️ React.js
* ⚡ Vite
* 🗂️ Redux (State Management)
* 🎨 CSS / Bootstrap / Material_UI
* 📡 Dummy API / JSON Server

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/mini-ecommerce-project.git
```

2. Navigate to the project folder:

```bash
cd mini-ecommerce-project
```

3. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
npm run dev
```

---

## 🌐 Environment Variables

This project does not require sensitive environment variables since it uses a dummy backend.

If needed in future, create a `.env` file:

```
VITE_API_URL=
```
